# Set Exercise 3: Page Layout

## Overview

This exercise recreates a gallery layout using modern CSS layout techniques.

The page arranges images and a video in a structured layout based on the provided screenshot.

## Techniques Used

- CSS Grid for arranging gallery items
- Responsive image and video scaling
- Semantic HTML structure

## Features

- Gallery items arranged in alternating large/small layouts
- Video element with playback controls
- Images cropped consistently while maintaining aspect ratio
