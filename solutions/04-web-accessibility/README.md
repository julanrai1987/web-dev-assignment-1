# Set Exercise 4: Web Accessibility

## Overview

This exercise improves the accessibility of a travel photography webpage.

The original page contained multiple accessibility issues including poor colour contrast, missing semantic structure, and limited support for screen readers and keyboard navigation.

## Accessibility Improvements

The following improvements were implemented:

- Semantic HTML structure using `header`, `main`, `section`, `article`, and `footer`
- Descriptive `alt` text for all images
- Accessible navigation using `<nav>`
- Buttons used for interactive actions (Add to basket)
- Checkout link labelled for screen readers
- Improved colour contrast for better readability
- Keyboard focus styles added for links and buttons
- Responsive images using CSS
- Skip link added for keyboard users

## Technologies Used

- HTML5 semantic elements
- CSS for layout and accessibility improvements

## Notes

The "Add to basket" buttons are included for accessibility demonstration but do not implement actual basket functionality.
