# Set Exercise 1: Semantic Markup

## Overview

This exercise recreates a recipe webpage using semantic HTML elements.

## Semantic Elements Used

- `header`
- `main`
- `section`
- `article`
- `h1` to `h3`
- `p`
- `ul` and `ol`

## Main Achievement From the This Exercise

The focus of this exercise was to structure the page content using meaningful HTML elements rather than relying on generic `div` elements.
